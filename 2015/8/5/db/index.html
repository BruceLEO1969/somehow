mycli

前几天从微博上发现了mycli这神器 ，一时感觉整个世界突然之前竟然充满了善意 。支持自动保全，mycli 支持语法高亮 ，还智能识别语句末尾的符号等等 ，极大的便利了 MySQL CLI 的操作 。当晚 在虚拟机的 Ubuntu Server 上试了一下 ，非常满意 ，恰逢手抖弄没了 mac 上的一堆数据 （User/ 下的数据都没了 ，新建用户不知道何故卡顿非常严重 ，重建索引并没有用），加上最近又忙又累无心细究原因以及解决方法 ，直接操起 U 盘将系统重新安装了一遍 ，一堆环境要重新部署 ，顺手就给本地也装了个 MySQL 以方便练习 ，没想到的是 ，装个数据库过程也不太顺利 。
